Privacy Policy for Alumni Mark

Effective Date: 7/2/25

Overview

Alumni Mark is a Chrome extension designed to help users manage and track alumni contacts in Salesforce. This privacy policy explains what information the extension collects, how it is used, and your rights regarding your data.

Information Collection and Use

  • Local Data Storage:
    • All user data—including uploaded CSV files, prospect information, and user progress—is stored locally within your browser using Chrome’s local storage API.
    • No data is transmitted to any external servers or third parties.
  • Data Types:
    • The extension processes the following data solely for its intended functionality:
      • Prospect information from user-uploaded CSV files (e.g., names, emails, job titles, company information, LinkedIn URLs).
      • User actions and progress (e.g., which prospects have been marked as alumni).
      • Scraped email addresses from Salesforce contact pages, stored only in your local browser session.
  • No External Transmission:
    • Alumni Mark does not send, upload, or share any data outside your local device.
    • The extension does not use analytics, tracking, or advertising services.

Permissions

  • activeTab, scripting, tabs, host_permissions: These permissions are required to allow the extension to interact with Salesforce pages, inject the floating panel, and automate updates. The extension only accesses the pages you are actively using it on.
  • storage: Used exclusively to save your data locally in your browser.

Data Security

  • All data is stored using Chrome’s local storage, which is only accessible to the extension and the user.
  • No data is accessible to the extension developer or any third party.

User Control

  • You may clear all stored data at any time using the “Clear CSV” button in the extension panel.
  • Uninstalling the extension will remove all locally stored data associated with Alumni Mark.

Children’s Privacy

  • Alumni Mark is not intended for use by children under the age of 16 and does not knowingly collect any personal information from children.

Changes to This Policy

  • Any changes to this privacy policy will be reflected in the extension’s documentation and update notes.

Contact
